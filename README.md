# Data Science Projects

## Apple Quality Classification
**Задача:**  
Определить качество яблок (хорошие/плохие) по характеристикам

**Что сделано:**  
- EDA: анализ данных, визуализация распределений
- Очистка: удаление выбросов, обработка пропусков
- Модели: 9 алгоритмов (Random Forest, Logistic Regression, деревья, KNN)
- Результаты: Random Forest показал accuracy 89.1%, AUC 0.955

**Технологии:** Pandas, Scikit-learn, Matplotlib
